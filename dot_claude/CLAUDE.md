# Global Claude Code Instructions

## Git Commits

- Always use conventional commits (e.g., `feat:`, `fix:`, `chore:`, `docs:`, `refactor:`, `test:`, `ci:`, `style:`, `perf:`, `build:`)
- Commit messages must be a single line with no body — no Co-Authored-By, no extra lines
- Format: `type(optional-scope): short description`
